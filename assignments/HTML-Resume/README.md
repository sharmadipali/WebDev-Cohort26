# HTML Resume Page

Live site:https://sharmadipali.netlify.app/

This project is a basic HTML resume webpage created as part of a learning assignment. It demonstrates the use of semantic HTML elements to structure a professional resume layout.

## Project Overview

The resume includes the following sections:

- Header with name and professional title
- About Me
- Skills
- Experience (in table format)
- Projects
- Education
- Contact Information
- Footer with copyright

## Technologies Used

- HTML5

## File Structure

/resume-project
│
├── index.html # Main resume webpage
└── README.md # Project documentation

## Features

- Semantic HTML tags (`header`, `main`, `section`, `article`, `footer`)
- Organized content using tables and lists
- Responsive viewport setup
- Clean and readable structure

## How to Run

1. Download or clone the project.
2. Open `index.html` in any web browser.
3. View your resume layout.
